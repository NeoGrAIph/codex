# Fork Documentation

This namespace contains fork-specific documentation that does not belong to the upstream/general
`docs/` surface.

Canonical layout:

- `features/<code-name>.md`: fork feature contract
- `projects/<code-name>/`: implementation dossier
- `research/<release>/`: release-specific baseline and gap analysis
- `audits/<code-name>.md`: findings-first documentation audit reports
